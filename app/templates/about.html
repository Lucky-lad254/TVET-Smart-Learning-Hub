About TVET Smart Learning Hub

Our Mission

To empower TVET (Technical and Vocational Education and Training) institutions and learners with an intelligent, modern learning management system that makes quality education accessible, engaging, and effective.

What We Offer

  • Comprehensive LMS: A complete platform for managing courses, materials, assessments, and student progress.
  • Intelligent Search: Find learning resources quickly with advanced search capabilities.
  • AI-Powered Learning: Get personalized help from an intelligent AI tutor.
  • Mobile-Friendly: Access learning materials from any device, anywhere, anytime.
  • Open Source: Built with open standards and freely available for TVET institutions.

Technology Stack

Built with modern, reliable technologies:

  • Backend: Python 3.12 with Flask framework
  • Database: SQLite (development), PostgreSQL-ready (production)
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Authentication: Secure password hashing with role-based access control
  • Search: Full-text indexing with OCR support
  • AI: Integrated with OpenAI-compatible APIs

Development Phases

Authentication, database, and dashboards for admin, teacher, and student roles.

Upload PDFs, Word docs, PowerPoints, videos, and audio. Organize by department, course, and module.

Full-text search, OCR for scanned documents, filters, and suggestions.

Lesson plans, assignments, quizzes, grading, and attendance tracking.

Progress dashboards, bookmarks, submissions, quiz attempts, and certificates.

AI tutor, auto quiz generation, concept explanation, and personalized recommendations.

Advanced analytics, reports, user management, and system configuration.

Testing, Docker deployment, GitHub Actions CI/CD, security hardening.
